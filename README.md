# NodeJS_API_Mongo

API RESTful desarrollada en Node.js para realizar operaciones CRUD sobre una base de datos MongoDB utilizando Express y Mongoose.

Este proyecto proporciona una estructura base para construir y extender un backend moderno, aplicando buenas prácticas de organización mediante rutas, controladores, modelos y configuración de base de datos. Es adecuado como proyecto académico o como punto de partida para servicios backend reales.

---

## Tecnologías utilizadas

- Node.js  
- Express.js  
- MongoDB  
- Mongoose  
- Middleware para manejo de peticiones y errores  

---

## Características principales

La API permite realizar las siguientes operaciones:

- Creación de registros  
- Consulta de registros  
- Actualización de registros  
- Eliminación de registros  

La estructura del proyecto separa responsabilidades de forma clara, facilitando el mantenimiento y la escalabilidad.

---

## Requisitos previos

Antes de ejecutar el proyecto, es necesario contar con:

- Node.js versión 14 o superior  
- npm  
- MongoDB (local o en la nube, por ejemplo MongoDB Atlas)
